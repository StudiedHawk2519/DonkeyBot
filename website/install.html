Get DonkeyBot

Make your homework bearable. Download the latest release package below and follow the easy integration guide for your browser.

Download Extension package (.zip)
Step 1: Download files. Get the latest code from GitHub by clicking the green button in the top-right corner and selecting "download zip".
GitHub Repository
Step 2: Unzip the package. Extract the downloaded .zip file and locate the chrome folder. You can safely delete the other files and folders, and store this folder somewhere permanent.
Step 3: Open Extension Configurations. Open a new browser tab, visit chrome://extensions in your URL bar, and toggle on Developer Mode in the top-right corner.
Step 4: Load Extension Panel. Click the Load unpacked button in the top-left corner, then select your extracted chrome folder to complete installation.
