Installation Guide

Step 1: Download files.

Get the latest code from the official GitHub repository. Download the entire 'chrome' folder - you should get a .zip file.

GitHub Repository
Step 2: Extract the .zip and keep the folder in a permanent location.
Step 3: Visit chrome://extensions and enable Developer Mode.
Step 4: Click Load unpacked and select the folder.
