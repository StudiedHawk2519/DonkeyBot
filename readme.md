## Installation Guide:

### Step 1: Download the Files

1. On this GitHub repository page, click the green **Code** button near the top right and select **"Download ZIP"**.
2. Extract the downloaded `.zip` file to locate the `extension` folder.
3. You can safely delete the other archive components and store this folder somewhere permanent.



### Chrome Instructions

1. Open a new browser tab and visit `chrome://extensions` in your URL bar.
2. Toggle on **Developer Mode** in the top-right corner.
3. Click the **Load unpacked** button in the top-left corner.
4. Select your extracted `extension` folder to complete installation.



### Firefox Instructions

1. Open a new browser tab, type `about:debugging#/runtime/this-firefox` into your URL bar, and hit Enter.
2. Click on **Load Temporary Add-on...**.
3. Navigate inside your extracted folder and choose the `manifest.json` configuration file to complete setup.
